TEAM Group — Confirmed Component Styles

Living reference doc. Each component is confirmed and deployed to employee_management_v2_dev. Test interactively below.

Date Picker

Confirmed

Native input[type="date"] with a red calendar icon and full-field click target. Clicking anywhere on the field opens the date picker — not just the icon.

  • Calendar icon is brand red (#EB1C2D) via CSS filter
  • ::-webkit-calendar-picker-indicator positioned absolute to cover entire field
  • Works on pre-filled (edit mode) fields — click anywhere to change
  • Read-only dates use a plain .read-only-box div instead, never a disabled input

Dual-List Picker

Confirmed

Replaces large lookup checkbox grids (100+ items). Available items on the left, selected on the right. Used for: ADP supervisor sites, Central Purchasing sites, BOSS cost sheet jobs, BOSS committees, Jonas job numbers.

  • Single click to highlight, double-click to move instantly
  • › / ‹ arrow buttons move all highlighted items
  • "Add visible" respects the current filter — only adds filtered items
  • Submits as a single hidden input[type="hidden"] with comma-joined value
  • Compatible with existing FormData.getAll(x).join(', ') server handlers
  • Container inline style is reset on render (clears old display:grid from replaced checkboxes)
