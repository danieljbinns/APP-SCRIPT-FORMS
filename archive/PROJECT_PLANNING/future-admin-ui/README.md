# ADMIN Folder

## Purpose
Future workflow design system - admin panel tooling for creating and managing workflows.

## Contents
- **Dashboard** - Admin interface for managing workflows
- **Templates** - Form templates library
- **WorkflowBuilder** - UI for creating workflows from existing tasks

## Status
**Future Development** - Not part of immediate deliverable

## Vision
This will eventually provide:
- Visual workflow builder
- Form template management
- Drag-and-drop workflow creation using pre-built tasks
- Form builder UI for creating new task forms

## Related To
Part of the larger initiative to enable multiple workflows and reusable task management across the organization.
