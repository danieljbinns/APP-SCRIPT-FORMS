DSS Learning Setup Form

Form Purpose:

This form handles DSS (training platform) learning setup for new hires.

Fields Include:

  • DSS Username
  • Training Modules Assigned
  • Learning Path Setup (New Hire tracks, Supervisor track, etc.)
  • Initial Training Complete status
  • DSS Notes

Workflow Position:

Parallel Execution: Triggers in parallel after Employee Setup Form (Form 2) completes
