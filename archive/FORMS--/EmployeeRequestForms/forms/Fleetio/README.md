# FORM_FLEETIO

## Description
Vehicle and fleet management tasks

## Setup
1. Create spreadsheet: `FORM_FLEETIO_DATA` in shared drive
2. Run: `clasp create --type webapp --title "FORM_FLEETIO"`
3. Update Config.gs with SPREADSHEET_ID
4. Run: `clasp push`
5. Deploy as web app

## Form Fields
- Task_ID
- Workflow_ID
- Employee_Name
- Vehicle_Assigned
- Vehicle_ID
- Fuel_Card_Issued
- Fuel_Card_Number
- Keys_Issued
- Fleet_Notes
- Completed_By
- Completion_Date
