# FORM_REVIEW306090

## Description
Performance review tracking at 30, 60, 90 days

## Setup
1. Create spreadsheet: `FORM_REVIEW306090_DATA` in shared drive
2. Run: `clasp create --type webapp --title "FORM_REVIEW306090"`
3. Update Config.gs with SPREADSHEET_ID
4. Run: `clasp push`
5. Deploy as web app

## Form Fields
- Task_ID
- Workflow_ID
- Employee_Name
- Review_Period
- Review_Date
- Performance_Rating
- Strengths
- Areas_For_Improvement
- Goals_Next_Period
- Reviewer_Name
- Completed_By
- Completion_Date
