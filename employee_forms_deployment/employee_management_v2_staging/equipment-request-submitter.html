Before You Begin

This form is for existing employees only. Use it to request new or additional system access or equipment for someone already in the system. Do not use this form for new hires — use the New Employee Request form instead.
When to use this form:
  • An employee needs access to an additional system (e.g. ADP, BOSS, Jonas)
  • A new computer, phone, or credit card is needed for an existing employee
  • An employee is taking over responsibilities and needs new system permissions
  • A Google Account needs to be created for an employee who didn't have one

Not sure which form? If the person hasn't started yet → New Employee Request. If they're changing roles or sites → Status / Position Change Request. If they're an active employee who just needs additional access → this form.

What Happens After You Submit

For your reference
You don't need to do anything further after submitting. Each team receives an email at the right time. You can check the current status at any time from the Dashboard in the TEAM Group Employee Portal using your Request ID.
1
IT Confirmation — Dave Langohr
Dave Langohr receives an email to review and confirm the request before IT Setup proceeds. This applies to all system and equipment access requests.
2
IT Setup
IT provisions all selected systems and equipment — email accounts, computer, phone, ADP access, BOSS, Jonas, and any other systems selected. You and the reporting manager receive a notification when complete.
3
Specialist Action Items (parallel)
Each relevant specialist team is notified simultaneously: Business Cards, Credit Card, Fleetio/Vehicle, and Jonas/Purchasing. These run in parallel and each team works independently.
⬦ Only the specialists matching your equipment and system selections are notified
✓
Request Complete
When all items are provisioned and specialist tasks are closed, you receive a completion notification. The employee now has access to everything that was requested.

Common Questions

The employee already has some access — can I add to it?
Yes. Select only the new access or equipment needed. Do not re-select things they already have — IT will add the new items without touching existing access.
Should I use this form or the Status / Position Change form?
If the employee is changing roles or sites, use the Status / Position Change Request — it handles role-level changes including updated ADP records, HR notification, and access adjustments as a package. Use this form only if you need to add specific access or equipment without a role change.
I selected the wrong system — can I cancel?
Once submitted, the request cannot be edited. Contact Dave Langohr or IT directly before provisioning begins — they can make corrections during the IT Confirmation step before anything is set up.
Which domain should I select for the Google Account?
Select the same domain as the reporting manager wherever possible. If the employee works across multiple entities, check with IT. The domain cannot easily be changed after the account is created.
What is WIS Assignment?
WIS (Work Instructions & Safety) is a BOSS module. After IT Setup, you as the reporting manager will receive a WIS action item email asking you to assign the appropriate WIS modules to the employee in BOSS.
I haven't received any confirmation or update emails — what do I do?
Check your spam folder first. If not there, log into the TEAM Group Employee Portal and check the workflow status yourself from the Dashboard using your Request ID. You can see the current stage and which step is pending.
