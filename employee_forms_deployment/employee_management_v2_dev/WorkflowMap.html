1. Initial Request

Hiring Manager submits request form.

2. ID Generation

System auto-generates Employee ID and DSS Username.

3. HR Verification

HR reviews details and confirms title/salary/manager.

4. IT Setup

IT creates email, assigns devices, and configures access.

Jonas Setup

Accounting sets up job numbers.

Credit Card

Finance team processes card request.

Fleetio

Fleet team sets up vehicle access.

Business Cards

Admin orders physical cards.

SiteDocs

Safety team adds worker to safety platform.

30/60/90

HR/Manager schedules performance reviews.

* Specialist steps run in parallel once IT Setup is submitted.
